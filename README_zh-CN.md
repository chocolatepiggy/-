## 🎮 3. 游戏机制

游戏共包含 **8 次航程**，每次航程分为 **4 个阶段**。

| 阶段 | 描述 |
|:---|:---|
| **阶段 1** | **港口采购**：在港口购买原材料或成品。 |
| **阶段 2** | **贸易交易**：完成客户订单以获取金币与声望。 |
| **阶段 3** | **维护与工资**：支付船只维护费与工匠薪资。 |
| **阶段 4** | **船只升级**：提升船只等级以降低后续运输成本。 |

### 📦 资源类型
- **原材料**
  - `麻布` – 基础服装原料。
  - `丝绸` – 高端商品面料。
  - `茶叶` – 制作香囊的奢侈饮品。
- **成品**
  - `麻衣` – 基础服装（织女制作）。
  - `布衣` – 中级服装（织女制作）。
  - `绫罗绸缎` – 高价值织物（纺织大师制作）。
  - `香囊` – 奢侈香囊（香囊师制作）。

### 👷 工匠系统
- **织女**：制作麻衣与布衣。薪资：8金币/回合。
- **纺织大师**：可制作麻衣、布衣与绫罗绸缎。薪资：12金币/回合。
- **香囊师**：仅制作香囊。薪资：20金币/回合。
- **熟练度提升**：工人累计生产足够物品后将升级，产出翻倍，但薪资临时增加。

### 💰 税收与财务
- **增值税**：约5%，针对成品利润征收。
- **所得税**：约10%，针对航程净利润征收。
- **运输费**：根据货物数量计算，扣除船只等级折扣。  
  公式：`max(5, 货物数量×2 - 船只等级×5)`

## 💡 5. 策略建议
1. **平衡开支**：切勿将所有金币用于采购，务必预留资金支付工资与维护费，避免破产。
2. **尽早升级**：船只升级能大幅降低长期运输成本，有盈余时优先升级。
3. **优化工匠**：解雇前确保工匠达到“熟练”状态，产出翻倍可大幅提升单回合回报率。
4. **商品选择**：成品利润高于原材料，但需计入增值税成本。
5. **税务规划**：接取高收益订单前，请提前估算所得税影响。

## 🛡️ 7. 故障排除
- **无法存档**：请确保项目文件夹具有写入权限，并检查杀毒软件是否拦截了 JSON 文件生成。
- **启动崩溃**：确认 Python 版本 ≥3.8。部分旧版系统可能缺少 `tkinter` 组件。
- **黑屏或界面异常**：更新显卡驱动；若因 DPI 缩放导致显示异常，请尝试以管理员身份运行。
